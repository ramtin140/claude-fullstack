# FIFA Soul — پلتفرم تورنمنت‌های فوتبال دیجیتال

یک اپلیکیشن فول‌استک برای برگزاری تورنمنت‌ها و مسابقات آنلاین فیفا، با طراحی الهام‌گرفته از قالب گرافیکی FIFA Soul (تم بنفش/طلایی، راست‌به‌چپ، فونت Vazirmatn).

## معماری

```
.
├── backend/   # Node.js + Express + SQLite (REST API, JWT auth)
└── frontend/  # React + Vite (RTL, Persian UI)
```

- **بک‌اند**: Express روی SQLite (`better-sqlite3`) با احراز هویت JWT، نقش‌های `member`/`admin`، و مسیرهای REST برای تورنمنت‌ها، مسابقات، اخبار، آمار، کاربران و خبرنامه.
- **فرانت‌اند**: React 18 + React Router با فونت متغیر Vazirmatn (سبک‌ترین فونت فارسی با پوشش کامل وزن‌ها در یک فایل) و آیکون‌های سبک `lucide-react`.
- **پنل مدیریت**: مسیر `/admin` (فقط برای نقش admin) برای مدیریت کامل تورنمنت‌ها، مسابقات، اخبار و کاربران.

## اجرا در محیط توسعه

### ۱. بک‌اند

```bash
cd backend
npm install
cp .env.example .env   # در صورت نیاز مقادیر را تغییر دهید
npm run dev             # http://localhost:4000
```

اولین اجرا به‌صورت خودکار دیتابیس SQLite را در `backend/data/fifasoul.db` می‌سازد و با داده‌های نمونه seed می‌کند.

کاربر ادمین پیش‌فرض: `admin@fifasoul.test` / `admin123`

### ۲. فرانت‌اند

```bash
cd frontend
npm install
npm run dev              # http://localhost:5173
```

درخواست‌های `/api/*` در حالت توسعه به‌صورت خودکار به بک‌اند (پورت ۴۰۰۰) پروکسی می‌شوند.

## امکانات

- ثبت‌نام / ورود کاربران با رمزنگاری bcrypt و توکن JWT
- مدیریت تورنمنت‌ها (لیگ/کاپ) و ثبت‌نام کاربران در آن‌ها
- مدیریت مسابقات با وضعیت زنده/در انتظار/پایان‌یافته
- بخش اخبار با دسته‌بندی (بازی‌های فعال، جدیدترین‌ها، گیک‌ها، آموزش‌ها)
- آمار خانه (رو در رو، پلی‌آف، لیگ، کاپ) و جدول اعضای برتر — همگی از دیتابیس واقعی محاسبه می‌شوند
- عضویت در خبرنامه با شماره موبایل
- پنل مدیریت کامل با CRUD برای تورنمنت‌ها، مسابقات، اخبار و کاربران

## نکته درباره مستندات

ریپازیتوری در ابتدا فقط یک قالب خالی Streamlit بود و مستنداتی به‌جز تصویر طراحی صفحه اصلی همراه درخواست ارسال نشده بود؛ بنابراین معماری، مدل داده و صفحات بر اساس تحلیل تصویر و بهترین شیوه‌های متداول برای این نوع پلتفرم طراحی شدند.
